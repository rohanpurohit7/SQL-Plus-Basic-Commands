# Oracle SQL*Plus And Pluggable Database Command Lab

This repository consolidates the useful SQL*Plus, SQL DDL/DML, and Oracle pluggable database administration commands from the former standalone SQL*Plus and PDB lab repositories into one sanitized command exercise.

## Contents

- [Narrative Exercise](docs/narrative-exercise.md)
- [ERD And Command Flow Diagrams](docs/erd-and-command-flow.md)
- [Consolidated SQL Script](commands/sqlplus-pdb-consolidated-lab.sql)
- [Command Catalog](commands/command-catalog.md)

## Scope

The guide covers:

- SQL*Plus session commands and output formatting
- SELECT queries against user and data dictionary views
- table creation, inserts, updates, deletes, commits, and drops
- constraints, indexes, views, synonyms, and sequences
- CDB/PDB connection checks and pluggable database open/status commands
- tablespace, datafile, redo log, control file, and archive mode inspection
- external table, materialized view, partitioning, and Data Pump command patterns

## Source Document Policy

Original Word source documents were removed. This repository keeps only extracted, sanitized, reusable database commands and explanatory diagrams.
